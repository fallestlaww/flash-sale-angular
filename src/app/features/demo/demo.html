Demo lab

Batch-fires POST /orders to visualize Self-Redis: atomic stock (oversell) and the fixed-window rate-limit. Needs a running backend and an existing event.

Oversell

N orders spread across K distinct users (avoids rate-limit). Expect ~stock × 201, the rest × 409 (atomic decrement on the backend).

Rate-limit

Spam N orders from ONE user (current X-User-Id). First ~20 pass (201/409), the rest → 429 (60s fixed window).
